Mathematical Formulation

1. Encoding (Evaluation Map)

An Algebraic Geometry (AG) code is defined by evaluating functions from a Riemann-Roch space $L(G)$ at a set of rational points $D = \{P_1, P_2, \dots, P_n\}$ on a curve $\mathcal{X}$. The codeword $\vec{c}$ is the evaluation map $ev: L(G) \to \mathbb{F}_q^n$.

2. Decoding (Syndromes & Dual Space)

The received vector $\vec{r} = \vec{c} + \vec{e}$ is checked against the dual code $C_\Omega(D, G)$, which relies on the parity-check matrix $H$ composed of differential forms. The syndrome vector $\vec{s}$ reveals the discrepancy $\Delta(P)$.
